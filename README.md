# 📝 Interactive To-Do List Application

A dynamic, interactive, and user-friendly To-Do List application built as a front-end portfolio piece. This project allows users to create custom categorized lists, manage specific tasks within each list, and preserve their data even after refreshing the page.

🌍 **Live Demo:** [View the website here](https://lexxy10101.github.io/TO_DO_LIST/)

---

## 🚀 Features

- **Categorized Lists:** Create multiple independent lists (e.g., Work, Groceries, Personal) using a clean pop-up modal view.
- **Task Management:** Click on any created list to open its unique space where you can add and track specific tasks.
- **Persistent Data (LocalStorage):** Uses browser storage to save your lists and tasks locally, ensuring you don't lose your progress when reloading or closing the tab.
- **Seamless Navigation:** Easily switch back and forth between the main dashboard and specific list views using an intuitive back-arrow navigation system.
- **Clean UI & Modal System:** Styled with a modern dark theme and interactive modal overlays for an optimized user experience.
- **Quick Deletion:** Easily delete entire lists or individual tasks instantly.

---

## 🛠️ Technologies Used

Built entirely with core front-end languages to focus on robust DOM manipulation and data persistence:

- **HTML5** – Semantically structured layout featuring multi-view dashboards and structured modal forms.
- **CSS3** – Custom responsive styling featuring absolute positioning, flexbox alignment layouts, visual overlay designs, and utility classes for state toggling (`.hidden`).
- **JavaScript (Vanilla JS)** – Powering the core application logic including state management, custom array methods (`filter`, `forEach`, `find`, `splice`), event propagation handling, and Web Storage API synchronization.
- **FontAwesome** – Scalable vector icons used for intuitive interface actions (plus, back-arrow, delete marks).
